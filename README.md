## 📦 What is agentplex?

agentplex is a software that helps you organize different tasks that depend on each other. Imagine a to-do list where some tasks must finish before others start. agentplex helps automate this, so things happen smoothly and correctly. It is good for handling complex plans that involve many steps and different processes.

This tool works on a system called a Directed Acyclic Graph (DAG). That means tasks have clear directions and no loops, so the order of work is logical and predictable. The program also keeps track of the current state of every task, so you always know what is done and what is next.

## 🎯 How to Use agentplex

### Opening the application

- Double-click the agentplex icon on your desktop or start menu.
- The main window will show workflows you create or load.

### Managing workflows

- **Create a new workflow:** Click the 'New Workflow' button. Here you can start adding tasks.
- **Add tasks:** Tasks are single steps in your workflow. Click 'Add Task' and name the task.
- **Set order:** Drag and drop tasks to arrange their sequence. This ensures tasks run one after another.
- **Run workflows:** Once your workflow is ready, press 'Start'. agentplex will run tasks in the order you set.

### Monitoring progress

- Watch your tasks turn green as they finish.
- If any task fails, agentplex will stop and show an error message.
- You can pause or stop at any time.
